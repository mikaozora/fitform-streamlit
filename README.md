
## Installation

#### Install fitform with pip

* Install opencv

```bash
  pip install opencv-python
```

* Install mediapipe

```bash
  pip install mediapipe
```

* Install numpy

```bash
  pip install numpy
```

* Install pygame

```bash
  pip install pygame
```

* Install streamlit

```bash
  pip install streamlit
```

## Run Code

* Go to the directory where you clone the code  

```bash
  streamlit run Home.py
```
